# 高分辨质谱仪分子式计算器

基于高分辨质谱测试数据（四位小数）的分子式计算工具。
输入精确质量后，程序会在给定误差范围内枚举可能的元素组成，并按化合物类型整理候选分子式。可以限制元素种类、个数。

## 计算原理

工具使用单同位素精确质量进行匹配。内置元素包括 C、H、N、O、S、P、F、Cl、Br、I 和 B，每个元素使用固定的精确质量值参与求和。

计算时，程序会从目标质量出发，在 `±0.002 Da` 的窗口内搜索元素组合。每一个候选式的理论精确质量会与输入质量比较，并给出质量偏差：

```text
质量偏差 = 理论质量 - 输入质量
ppm 偏差 = 质量偏差 / 输入质量 * 1,000,000
```

为了减少明显不合理的结果，程序会做基础化学约束筛选，例如要求至少含有 C 和 H，并用氢原子数量上限排除一部分价态明显不合理的组合。结果会按 CHO 化合物、硼化合物、卤素化合物、硫/磷化合物、含氮化合物和其他化合物分类，并在每一类中按绝对质量误差从小到大排序。

如果勾选 `[M+Na]` 离子峰，程序会先从输入质量中减去 Na 的质量 `22.989769 Da`，再按中性分子的精确质量进行分子式搜索。

## 如何使用

1. 打开页面：<https://sheephjc.github.io/tools/MassCalculator/>
2. 在输入框中填写高分辨质谱得到的精确质量，例如 `180.0634`。
3. 根据需要设置元素的最小值和最大值，或选择 `[M+Na]` 离子峰。
4. 点击“计算”，查看候选分子式、理论质量、杂原子数、Da 偏差和 ppm 偏差。

## 应用场景

- 根据 HRMS 精确质量快速生成候选分子式。
- 对合成产物、天然产物或反应副产物进行初步组成判断。
- 在已知元素范围时，通过限制 C、H、O、N、S、P、卤素、B 等元素数量缩小搜索空间。
- 对疑似钠加合峰 `[M+Na]` 进行中性分子质量换算和候选式筛选。

## 注意事项

- 结果只是候选分子式，不等同于结构确认。
- 建议结合 MS/MS 碎片、同位素峰型、氮规则、样品背景和反应路径进一步判断。
- 当前页面支持的输入质量范围约为 `16.0313-1000 Da`。
- 搜索空间过大时可能超时，此时应增加元素数量限制或明确可能存在的杂原子。
